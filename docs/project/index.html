Projects

This section stores some of my ongoing projects involving R programming.

Please note, some projects in are still in progress...

Labor certification processing time analysis

Short project evaluating labor certification processing time for PERM/Green Card Holders This project investigates whether the Trump administration has slowed down the green card application process for foreign workers?. The analysis relies on DOL/OFLC and USCIS open data. Results Green Card Processing time under Trump - Brief Green Card Processing time under Trump - Github Repo

Linkedin data analysis

Study of jobs and competitivity of cities This mini-study project was a take-home test I completed in winter 2018 as part of a job interview for a consultancy with IFC (in collaboration with LinkedIn). It used it as an opportunity to refresh some R programming and learn to use the xaringan package to make html presentations. Results Linkedin Analysis Presentation Linkedin Analysis Github Repo

NYE fundraising data analysis

Analysis of Fundraising data for Non Profit “NYE Encounter” Since March 2018, I have worked part-time as a Pro-Bono Fundraising Data Analyst for the nonprofit organization New York Encounter. I performed several ETL tasks –via SQL-based data transformations and R programming– to set up the integration of data that existed in separate and disconnected platforms. Namely, the Nonprofit had informations on donors and attendees on SendGrid, donations recorded via Square and miscellaneous other information in Salesforce.

World Development Indicator data analysis

Study of World Bank’s WDI indicator Life expectancy at birth This mini-study project was an exercise to learn how to use the World Bank’s API for the World Development Indicator (WDI) repository. In this study, I explore the indicator Life expectancy at birth and I also do some comparative analisys using other indicators that I expect to be highly correlated with life expectancy: namely GDP and the status of Fragile and conflict affected situations (FCS).

IADB Survey of WSS suppliers in Brazil 2018/2019

Please note, the survey is still in progress, hence the results are subject to change and NOT (yet) ready for dissemination. Link to Project Page https://lulliter.github.io/BrazilWaterSurvey/ Purpose of the study The survey is part of a larger effort to collect original data in preparation of the publication “Development in America” 2020 that will focus on Infrastructure (Energy, Water and Transportation) sector in LAC.
